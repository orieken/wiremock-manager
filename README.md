# wiremock-manager


## Trying it out locally

* `wiremock-manager`

## Command line options

* `--port`: Set the HTTP port number e.g. `--port 9999`
* `--root-dir`: Sets the root directory, under which mappings and __files reside. This defaults to the current directory.

## Testing

* `npm test`
